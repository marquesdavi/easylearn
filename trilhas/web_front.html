Criando sua Primeira Página com HTML

Criar uma página web é como construir uma casa digital, onde cada elemento HTML é uma parte importante da estrutura. Aprenda passo a passo como montar sua primeira página HTML e comece a explorar o mundo do desenvolvimento web!

Elemento HTML Descrição
<!DOCTYPE html> Define a versão HTML e o tipo de documento.
<html> Define o início do documento HTML.
<head> Contém metadados e informações sobre o documento.
<title> Define o título da página exibido na aba do navegador.
<body> Contém o conteúdo visível da página, como textos, imagens e outros elementos.

Criando o Esqueleto Básico

Comece definindo o tipo de documento, o início do HTML, o cabeçalho e o corpo da página. 

                            
        <!DOCTYPE html>
        <html>
        <head>
            <title>Minha Primeira Página</title>
        </head>
        <body>
            <h1>Bem-vindo à Minha Página</h1>
            <p>Esta é a minha primeira página HTML!</p>
        </body>
        </html>

Adicionando Elementos

Explore diversos elementos HTML para adicionar texto, imagens, links e listas à sua página. 

                            
        <!DOCTYPE html>
        <html>
        <head>
            <title>Minha Primeira Página</title>
        </head>
        <body>
            <h1>Bem-vindo à Minha Página</h1>
            <p>Esta é a minha primeira página HTML!</p>
            <img src="caminho_para_sua_imagem.jpg" alt="Minha Imagem">
            <a href="https://www.exemplo.com">Visite Exemplo.com</a>
            <ul>
                <li>Item 1</li>
                <li>Item 2</li>
                <li>Item 3</li>
            </ul>
        </body>
        </html>
