Decisões Condicionais em JavaScript

As decisões condicionais em JavaScript permitem que você tome diferentes caminhos de execução de acordo com as condições definidas. Aprenda como utilizar as estruturas if, else-if, else e métodos úteis para fazer verificações em seus programas JavaScript.

Estrutura if

A estrutura if permite executar um bloco de código se uma condição especificada for verdadeira. 

                
let idade = 18;
if (idade >= 18) {
    console.log("Você é maior de idade.");
}

Estrutura else-if

A estrutura else-if permite verificar várias condições diferentes para executar blocos de código diferentes. 

                
let hora = 14;
if (hora < 12) {
    console.log("Bom dia!");
} else if (hora < 18) {
    console.log("Boa tarde!");
} else {
    console.log("Boa noite!");
}

Estrutura else

A estrutura else é executada se nenhuma das condições anteriores for verdadeira. 

                
let diaDaSemana = "Sábado";
if (diaDaSemana === "Sábado" || diaDaSemana === "Domingo") {
    console.log("É final de semana!");
} else {
    console.log("É dia de semana.");
}

Métodos úteis para verificação

Além das estruturas condicionais, existem métodos úteis para verificar valores em JavaScript, como isNaN() para verificar se um valor não é um número, e includes() para verificar se uma string contém outra string. 

                
let numero = "10";
if (!isNaN(numero)) {
    console.log("É um número válido.");
} else {
    console.log("Não é um número válido.");
}

let frase = "Olá, mundo!";
if (frase.includes("Olá")) {
    console.log("A frase contém a palavra 'Olá'.");
} else {
    console.log("A frase não contém a palavra 'Olá'.");
}
